# Welcome to Device Portal

Device Portal is a React-based library that simplifies peer-to-peer data sharing using WebRTC.

## Packages
- **[@device-portal/react](https://www.npmjs.com/package/@device-portal/react)**: The React client library.
- **[@device-portal/server](https://www.npmjs.com/package/@device-portal/server)**: The signaling server for WebRTC negotiation.

## Core Features
- **Easy Setup**: Use `DevicePortalProvider` and `DevicePortalConsumer` to establish connections.
- **WebRTC Data Channels**: Reliable, low-latency data transfer between devices.
- **React Hooks**: Intuitive hooks for managing peer states and data flow.

## Getting Started
Explore the components in the sidebar to see how Device Portal works in different scenarios.

- **Example Components**: Basic usage of the provider and consumer.
- **Example Hooks**: Low-level hooks for custom implementations.
- **Cookie Clicker**: A fun demonstration of real-time state synchronization.
